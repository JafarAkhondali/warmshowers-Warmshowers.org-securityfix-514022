INSTALLATION INSTRUCTIONS
=========================
Follow the following steps:

- Go to http://www.osmobi.com
- Create an account and login
- Create a new project. 
- Install the osmobiclient module and write down the security key
- Use the security key to create your project at osmobi
- Make your website mobile and launch it

For more installation instructions go to the Osmobi support pages (http://osmobi.zendesk.com)
